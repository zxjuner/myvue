#### 计算属性
1、计算属性缓存 VS 方法
*计算属性*基于他们的依赖进行缓存，只有在它的相关依赖发生改变时才会重新求值。

相比之下，每当触发重新渲染时，调用*方法*将总会再次执行函数。

#### Vue 指令
1、指令是带有 `v-` 前缀的特殊特性。指令特性的值预期是*单个 JavaScript 表达式*，指令的指责是，当表达式的值改变时，将其产生的连带影响响应式地作用于 DOM。

2、`v-bind` 缩写
```
<!-- 完整语法 -->
<a v-bind:href="url">...</a>

<!-- 缩写 -->
<a :href="url">...</a>
```

3、`v-on` 缩写
```
<!-- 完整语法 -->
<a v-on:click="doSomething">...</a>

<!-- 缩写 -->
<a @click="doSomething">...</a>
```
